# 2883. Drop Missing Data

```
DataFrame students
+-------------+--------+
| Column Name | Type   |
+-------------+--------+
| student_id  | int    |
| name        | object |
| age         | int    |
+-------------+--------+
```

There are some rows having missing values in the `name` column.

Write a solution to remove the rows with missing values.

The result format is in the following example.

&#x20;

**Example 1:**

<pre><code><strong>Input:
</strong>+------------+---------+-----+
| student_id | name    | age |
+------------+---------+-----+
| 32         | Piper   | 5   |
| 217        | None    | 19  |
| 779        | Georgia | 20  |
| 849        | Willow  | 14  |
+------------+---------+-----+
<strong>Output:
</strong>+------------+---------+-----+
| student_id | name    | age |
+------------+---------+-----+
| 32         | Piper   | 5   |
| 779        | Georgia | 20  | 
| 849        | Willow  | 14  | 
+------------+---------+-----+
<strong>Explanation: 
</strong>Student with id 217 havs empty value in the name column, so it will be removed.
</code></pre>



## Solution

```python
import pandas as pd

def dropMissingData(students: pd.DataFrame) -> pd.DataFrame:
    result = students.dropna(subset=["name"]) #dropna drops all nan values
    return result
```
